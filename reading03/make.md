TLDR - make
==========

Overview
--------

make - a command which relies on a makefile to compile

Examples
--------

-make clean-
	
	-removes all made files

-Change directory before reading -
	
	-C

-Print Debugging Information -

	-d

-Use a specific Makefile -

	-f [FILE]
