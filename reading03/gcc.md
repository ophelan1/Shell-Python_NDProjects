TLDR - gcc
==========

Overview
--------

gcc - a compiler for c

Examples
--------

-run compiler with debugging-
	
	-g

-run compiler with user privileges-
	
	-pg

-run compiler to output specific name -

	-o [NAME]

-compile to object file-

	-c
