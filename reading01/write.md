TLDR - write
==========

Overview
--------
	
	write - send a message to another user. After being entered, will copy all input to the 
			  terminal until an EOF character is reached is reached. 

	>>> write USER [TERMINAL]


		>>> write Owen
		> (Sends a message to Owen)


EOF
---

	On my mac, it's ctrl-C
