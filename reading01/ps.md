TLDR - ps
==========

Overview
--------

[ps (modifier)] shows current process(es)

Examples
--------
- See only the RUNNING processes -

	>>> ps -r

- See every process on the system -

	>>> ps -e

- See every process associated with current terminal - 

	>>> ps T

- See all processes except the session leader -
	
	>>> ps -d
