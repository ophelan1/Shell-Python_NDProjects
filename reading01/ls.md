TLDR - ls
==========

Overview
--------

[ls (modifier) [DIRECTORY]] lists the contents of a directory. 

Examples
--------

-List all entries including hidden ones-
	
	>>> ls -a [DIRECTORY PATH]

-List the long form of each item-
	
	>>> ls -l [DIRECTORY PATH]

-List contents in order of latest modified-
	
	>>> ls -t [DIRECTORY PATH]
