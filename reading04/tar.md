TLDR - tar
==========

Overview
--------

	tar - The GNU version of the tar archiving utility

Examples
--------

-Standard Operations-

	x - extract 
	u - update
	c - create
	r - append

-Specify Options, follow the Operations-
	
	v - verbose 
	f - file F
	z - gzip
