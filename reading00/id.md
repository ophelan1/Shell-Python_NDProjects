TLDR - id
==========

Overview
--------

[id (modifier) (USER)] prints the id of the USER, or if there is no USER specefied, the current user.

Examples
--------

-id- the current user:
	
	id

-name- id, instead of number:
	
	id -n (USER)

-group- id:
	
	id -g (USER)
